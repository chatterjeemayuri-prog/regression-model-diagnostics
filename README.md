# regression-model-diagnostics
Robust Linear Regression and Model Diagnostics

## Project Overview
This project demonstrates linear regression modelling and diagnostic analysis using R. The goal is to evaluate model assumptions, detect influential observations, and interpret regression results in a practical statistical workflow.

## Techniques
- Linear regression
- Residual diagnostics
- Heteroskedasticity checks
- Influence measures
- Visualization of model assumptions

## Tools
- R
- ggplot2
- lm()
