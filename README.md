# regression-model-diagnostics
Robust Linear Regression and Model Diagnostics

## Project Overview
This project demonstrates linear regression modelling and diagnostic analysis using R. The goal is to evaluate model assumptions, detect influential observations, and interpret regression results in a practical statistical workflow.

## Techniques
- Linear regression
- Residual diagnostics
- Heteroskedasticity checks
- Influence measures
- Visualization of model assumptions

## Tools
- R
- ggplot2
- lm()

## How to Run

1. Open the R script `regression_diagnostics.R`
2. Run the script in R or RStudio
3. The script will:
   - Simulate a dataset
   - Fit a linear regression model
   - Produce diagnostic plots
   - Display regression summary results

## Key Statistical Concepts Demonstrated

This project demonstrates several core statistical modeling and diagnostic techniques:

- Linear regression modelling
- Residual diagnostics and model assumption checks
- Influence analysis and leverage points
- Interpretation of regression coefficients
- Practical statistical reasoning using R
